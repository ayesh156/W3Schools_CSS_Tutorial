column-fill

The column-fill proeprty specifies how to fill columns, balanced or not.

Tip: If you add a height to a multi-column element, you can control how the content fills the columns. The content can be balanced or filled sequentially.

Syntax :- column-fill: balance|auto|initial|inherit;

balance :- Default value. Fill each column with about the same amount of content, but will not allow the columns to be taller than the height (so, columns might be shorter the height as the browser distributes the content evenly horizontally.

auto :- Fills each column until it reaches the height, and do this until it runs out of content (so, this value will not necessarily fill all the columns nor fill them evenly)

Specify how to fill columns:

column-fill: auto;
