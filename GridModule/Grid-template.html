grid-template

The grid-template property is a shorthand property for the following properties:

  • grid-template-rows
  • grid-template-columns
  • grid-template-areas

Syntax :- grid-template: none|grid-template-rows / grid-template-columns|grid-template-areas|initial|inherit;
