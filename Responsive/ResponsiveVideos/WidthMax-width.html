Using The width Property

If the width property is set to 100%, the video player will be responsive and scale up and down

Resize the browser window to see how the size of the video player will scale.

Notice that in the example above, the video player can be scaled up to be larger than its original size. A better solution, in many cases, will be to use the max-width property intead.

Using The max-width Property
