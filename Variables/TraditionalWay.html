Variables

The var() function is used to insert the value of a CSS variable.

CSS variable have access to the DOM, which means that you can create variable with local or global scope, change the variable with JavaScript, and change the variables based on media queries.

A good way to use CSS variable is when it comes to the colors of your design. Instead of copy and paste the same colors over and over again, you can place them in variables.
