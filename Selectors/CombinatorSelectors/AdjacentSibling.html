Combinators

A CSS selector can contain more than one simple selector. Between the simple selector, we can include a combinator.

There are four different combinators in CSS:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selelctor (+)
  • general sibling selecotr (~)

Adjacent Sibling Selector

The adjacent sibling selector is used to select an element that is directly after another specific element.

Sibling element must have the same parent element and "adjacent" means "immediately following".
