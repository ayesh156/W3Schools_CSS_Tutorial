The :valid selector selects form elements with a value that validates according to the element's settings

Note: The :valid selector only works for form elements with limitations, such as input elements with min and max attributes, email fields with a legal email, or number field with a numeric value, etc

Select and style only if the value of the (input) element is valid

Try typing an illegal e-mail address, to see the styling dissapear

Try typing a number out of range (between 5 to 10), to see styling dissapear
