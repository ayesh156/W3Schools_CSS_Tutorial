The text-shadow property adds shadow to text.

This property accepts a comma-separated list of shadows to be applied to the text.

Synatx - text-shadow: h-shadow v-shadow blur-radius color|none|intial|inherit

  • h-shadow - Required. The position of the horizontal shadow. Negative value are allowed
  • v-shadow - Required. The position of the vertical shadow. Negative values are allowed
  • blur-radius - Optional. The blur radius. Default value is 0
  • color - Optional. The color of the shadow.
