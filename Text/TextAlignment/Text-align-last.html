The text-align-last property specifies how to align the last line of a text:

That the text-align-last property sets the alignment for all last lines within the selected element. So, if you have a (div) with three paragraphs in it, text-align-last will apply to the last line of EACH of the paragraph. To use text-align-last on only the last paragraph in the container, you can use :last child, see example below.

Note: In Edge prior 79, the text-align-last property only works on text that has "text-align: justify".

Align the last line of text in three (div) elements:

text-align-last: right;
