Top Property

The top property affects the vertical position of a positioned element. This property has no effect on non-positioned element.

  • If position: absolute; or position: fixed; - the top property sets the top edge of an element to a unit above/below the top edge of its nearest positioned ancestor.
  • If position: relative; - the top property makes the element's top edge to move above/below its normal position.
  • If position: sticky; - the top property behaves like its postion is relative when the element is inside the viewport, and like its position is fixed when it is outside
  • If position: static; - the top property has no effect.

Set the top edge of the positioned (div) element 50px down from the top edge of its nearest positioned ancestor:

This div element has position: absolute, and top: 0.
This div element has position: absolute, and is placed 50 pixels down from the top edge of the containing positioned element.

Use the top property with a negative value and for an element with no positioned ancestors:

This div element has position: relative;
This div element has position: absolute and top: -20px.
This div element has no parent element, has position: absolute and is placed 800 pixels down from the top of the document body.
