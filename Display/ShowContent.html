Using CSS together with JavaScript to show content

This example demonstrates how to use CSS and JavaScript to show an element on click.

Click to show panel

This panel contains a div element, which is hidden by default (display: none).

It is styled with CSS and we use JavaScript to show it (display: block).

How it works: Note that the p element with class="flip" has an onclick attribute attached to it. when the user clicks on the p element, a function called myFunction() is executed, which changes the style of the div with id="panel" from display:none (hidden) to display:block (visible).
