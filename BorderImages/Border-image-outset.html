border-image-outset Property

The border-image-outset property specifies the amount by which the border image area extends beyond the border box.

Syntax :- border-image-outset: length|number|initial|inherit

Note: The border-image-outset property can take from one to four values (top, right, bottom and left sides). If the fourth values is omitted, it is the same as the second. If the third one is also omitted, it is the same as the first. If the second one is also omitted, it is the same as the first.
