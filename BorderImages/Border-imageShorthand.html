Border-image Shorthand

The border-image property allows you to specify an image to be used as the border around an element.

The border-image property is a shorthand property for:

  • border-image-source
  • border-image-slice
  • border-image-width
  • border-image-outset
  • border-image-repeat

Omitted values are set to their default values.

Syntax :- border-image: source slice width outset repeat|initial|inherit

Specify an image as the border around an element:

The border-image Property - repeat

Here the middle section of the image are repeated to create the border:

border-image: url(border.png) 30 round;

CSS border-image - Different Slice Values

Different slice values completely changes the look of the border:

border-image: url(border.png) 50 round;

border-image: url(border.png) 20% round;

border-image: url(border.png) 30% round;

Note: Internet Explorer 10, and earlier versions, do not support the border-image property.
