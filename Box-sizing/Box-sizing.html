box-sizing

The box-sizing property defines how the width and height of an element are calculated: should they include padding and borbers or not.

Syntax :- box-sizing: content-box|border-box|initial|inherit;

content-box :- Default. The width and height properties (and min/max properties) includes only the content. Border and padding are not included

border-box :- The width and height properties (and min/max properties) includes content, padding and border

Include padding and border in the element's total width and height:

box-sizing: content-box (default):

Wdith and height only apply to the content of the element:

This div has a width of 300px. But the full width is 300px + 20px (left and right border) + 60px (left and right padding) = 380px!

box-sizing: border-box:

Width and height apply to all parts of the element: content, padding and borders:

Here, the full width is 300px, no matter what!
