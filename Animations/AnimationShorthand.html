animation Shorthand

The animation property is a shorthand property for:

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-fill-mode
  • animation-play-state

Note: Always specifiy the animation-duration property, otherwise the duration is 0, and will never be played.

Syntax :- animation: name duration timing-function delay iteration-count direction fill-mode play-state;

Binding an animation to a (div) element, using the shorthand property;

Move an element from 0px to 200px left. The animation lasts for 5 seconds. It then starts over again, and go on forever (infinite).
